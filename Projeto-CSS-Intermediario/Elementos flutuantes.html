Páginas flutuantes

O Float é um atributo que permite a div ser um elemento flutuante. É possível botar as opções:

  • Left
  • Right
  • None
  • inherit

Utilizar o comando Clear para que o Float faça parte do elemento. No comando clear, há as seguintes opções:

  • Left
  • Right
  • Both
  • None
