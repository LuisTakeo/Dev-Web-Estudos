Formatando Links

Os links possuem mais de um estado, a qual é possível a formatação de cada um deles. Os estados do link são:

  • a:link - Link a qual nunca foi visitado.
  • a:visited - Link a qual já foi visitado.
  • a:hover - Estado do link quando o mouse está em cima dele.
  • a:active - Link ativo, ou seja, quando seguramos o clique nele.
