Inline, block e Inline block

São atributos de display, aonde o conteúdo vai se posicionar de uma forma diferente na tela.

  • Inline: O conteúdo somente vai ocupar o espaço utilizado na tela. Quando tiver um novo conteúdo de mesmo tipo, vai ocupar o espaço disponível ao lado.
  • Block: O conteúdo vai ocupar todo espaço em bloco na tela, como acontece no h1, p, tr. Quando tiver um novo conteúdo, ele vai para o espaço de baixo.
  • Inline-block: Vai ocupar o espaço disponível e quando não tiver, vai ocupar o espaço abaixo
